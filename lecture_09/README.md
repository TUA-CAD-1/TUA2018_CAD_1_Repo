## ライノモデルとの連動
- 使用コンポーネント："Geometry"
　GH UI上でマウス左ダブルクリックで使用コンポーネント検索、または
　GH UI タブからドロップ

- RHデータとGHコンポーネントのリンク
　コンポーネント上でマウス左クリックで"Set One Geometry"または"Set Multiple Geometories"
　を選択し、続けてRH上でリンクしたいモデルを選択
