## モデリング実習（好きな椅子）
- 既存の、あるいは想像の好きな椅子を作ること。
- 寸法的に椅子として成り立っていること（座れない椅子は×）
- 薄い素材であっても平たいサーフェースではなく、厚みをつけること。
- 地面に立っていること（構造的に成り立っていること）。
- 質問は自由にすること（Google Classroom上でも可）。
- モデリングファイル（3dm）と立面図、上面図、レンダリング画像をそれぞれ一枚以上、来週月曜日までに提出すること。
